Core Concept

What is a Function?

A function is a rule that assigns exactly ONE output to each input.

output = f(input) or y = f(x)

Think of it as a machine:
Put in x → machine processes → get y

Examples:
• Cost = f(quantity): C = 50x
• Fare = f(distance): F = 13 + 1.5(d−4)
• Tax = f(income): T = 0.12 × income

Modeling

Mathematical Models

A mathematical model is a simplified representation of a real situation using equations, graphs, or tables.

📈 Revenue Model

R = price × quantity
R = 120x
(₱120 per item, x items sold)

💸 Cost Model

C = fixed + variable
C = 5000 + 80x
(₱5K rent + ₱80/item to produce)

💰 Profit Model

P = Revenue − Cost
P = 120x − (5000 + 80x)
P = 40x − 5000

⚠️ Model Limitations

Models are SIMPLIFICATIONS. They assume constant prices, unlimited demand, no competition, etc. Real life is messier!

Key Concept

Break-Even Analysis

Break-even point = where Revenue = Cost (profit = 0). Below this, you lose money. Above, you profit.

Break-even: Revenue = Cost
Price × x = Fixed + Variable × x
x = Fixed Cost / (Price − Variable Cost)

Example:
Price = ₱120/item, Fixed = ₱5,000, Variable = ₱80/item
x = 5000 / (120−80) = 5000/40 = 125 items

Interpretation

• Sell < 125 items → LOSS
• Sell exactly 125 → break even (₱0 profit)
• Sell > 125 → PROFIT

At 125 items:
Revenue = 120 × 125 = ₱15,000
Cost = 5000 + 80(125) = ₱15,000 ✓

At 200 items:
Profit = 40(200) − 5000 = ₱3,000 🎉

Linear vs Exponential

Interpreting Graph Shapes

Linear: y = mx + b

m = slope (rate of change)
b = y-intercept (starting value)

Straight line. Constant growth.
Example: Fare = 13 + 1.5d
Slope = ₱1.50/km

Exponential: y = a(b)ˣ

a = initial value
b = growth factor

Curved line. Accelerating growth.
Example: A = 10000(1.05)ᵗ
Compound interest growth

Slope interpretation: In y = 120x, the slope 120 means "revenue increases ₱120 for each additional item sold." Understanding slope = understanding the rate of change in your model.
