Core Concept

What is a Pattern?

A pattern is any regularity that can be recognized, described, and predicted.

Patterns can be found in:

  • Numbers (1, 3, 5, 7, 9...)
  • Shapes (⬡ ⬡ ⬡ repeated)
  • Events (schedule, routine)
  • Data (sales trends, seasons)

Why Do Patterns Matter?

🔮

Prediction

If we see a pattern, we can predict what comes next

🗣️

Communication

Patterns let us describe complex ideas simply

💡

Decision-Making

Recognizing patterns helps us make better choices

Key Concept

What is a Variable?

A variable is a symbol (usually a letter) that represents a value that can change.

Daily Budget Example

Let B = your daily baon
Let L = lunch cost
Let M = meryenda cost
Remaining = B − L − M

If B = ₱150, L = ₱60, M = ₱30
Remaining = 150 − 60 − 30 = ₱60

Distinction

Expressions vs. Equations

Expression

A combination of numbers, variables, and operations. No equals sign.

3x + 5

P × Q

B − L − M

Equation

A statement that two expressions are equal. Has an equals sign (=).

3x + 5 = 20

C = P × Q

R = B − L − M

Application

Patterns Help Us Decide

Scenario: You want to buy a ₱3,000 pair of shoes. You can save two ways:

Plan A: Fixed ₱200/week

Week 1: ₱200 | Week 2: ₱400 | Week 3: ₱600...
Pattern: 200n (arithmetic)
Reach ₱3,000 in 15 weeks

Plan B: Start ₱50, double each week

Week 1: ₱50 | Week 2: ₱100 | Week 3: ₱200...
Pattern: 50 × 2^(n-1) (geometric)
Reach ₱3,000+ by week 6!

Key Insight

Geometric growth is much faster than arithmetic growth. This is why compound interest is so powerful!

But Plan B requires ₱50+100+200+400+800+1600 = ₱3,150 saved (much more total money!)

Critical Thinking

Using Patterns to Compare

📱 Phone Plan A

₱299/month, 8GB data

Month 1₱299
Month 2₱598
Month 6₱1,794
Month 12₱3,588

Pattern: 299n (linear)

📱 Phone Plan B

₱199/month, 5GB + ₱15/extra GB

If you use 5GB₱199
If you use 8GB₱244
If you use 10GB₱274
If you use 15GB₱349

Pattern: 199 + 15(g−5) if g>5

Which is better? Depends on your data usage pattern! If you use ≤8GB, Plan B is cheaper. If you always use 8GB+, Plan A gives certainty.
